ac-coffee
=========

Emacs auto-complete source for CoffeeScript

Requirements
--------

* Node.js
* Coffee
* Emacs 24+

Installation
--------

0. Wait for me to complete my work
1. Clone this repo
2. Configure ```.emacs```

```lisp
;; My Coffee AC Source
(add-to-list 'load-path "path/to/ac-coffee/")
(require 'ac-coffee)
```

3. Done!
